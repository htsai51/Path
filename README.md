## Overview

Simple Path Finding C++ project with:

- cmake
- googletest
- A star algorithm

## Installation

- Checkout the repo (and submodules)
```bash
$ git clone --recursive https://github.com/htsai51/path.git
cd path
mkdir -p build
cd build
cmake ..
make
```

## Testing the code

- In your build/ directory
```bash
./test/cpp-test
```
